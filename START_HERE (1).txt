🎯 3 ВАРИАНТА РАСПРОСТРАНЕНИЯ:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ВАРИАНТ 1: Папка на флешке (ПРОСТЕЙШИЙ)
   1. Переименуй dist → BLUR_Portable
   2. Скопируй на флешку (1.5GB)
   3. Люди открывают папку и запускают START_BLUR.bat

ВАРИАНТ 2: ZIP архив (ПОПУЛЯРНЫЙ)
   1. Упакуй dist в ZIP
   2. Назови BLUR_Portable_v1.0.zip
   3. Загружай на облако/сайт (1.5GB)
   4. Люди скачивают и распаковывают

ВАРИАНТ 3: MSI инсталлятор (ПРОФЕССИОНАЛЬНЫЙ)
   1. Скачай NSIS https://nsis.sourceforge.io/
   2. Используй NSIS_INSTALLER.nsi
   3. Собери BLUR_Setup.exe
   4. Люди запускают installer

1. Размер BLUR.exe (~1.5GB)
   ✓ Это нормально (Python 500MB + spaCy 400MB + библиотеки)
   ✓ Не могу сделать меньше без потери функций
   ✓ После первого запуска работает быстро

2. Антивирусы могут ругаться
   ⚠️ PyInstaller exe часто false positive
   ⚠️ Добавь в исключения своего антивируса
   ⚠️ Или распространяй ZIP вместо exe

3. Первый запуск медленнее
   ✓ 10-15 сек - это нормально
   ✓ Следующие запуски быстрее
   ✓ Зависит от скорости диска

4. Требования
   ✓ Windows 10/11 (64-bit)
   ✓ 2GB RAM минимум
   ✓ 2GB свободного места

  - Используй spacy small модель

  - Сжми с UPX (осторожнее с антивирусом)

✓ Добавить digital signing:
  - Подпиши BLUR.exe сертификатом
  - Антивирусы будут меньше ругаться
  - Требуется код signing certificate

✓ Создать auto-updater:
  - Проверяй версии на сервере
  - Скачивай обновления автоматически
  - Обновляй BLUR.exe без вмешательства

✓ Собрать 32-bit версию:
  - Нужен отдельный Python 32-bit
  - PyInstaller на 32-bit машине
  - Больше не нужно (все 64-bit)
